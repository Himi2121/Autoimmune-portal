# 🧬 ImmunoTracks: Translational Autoimmune Portal

A systems biology analytics dashboard and diagnostic machine learning engine designed to bridge raw high-throughput cellular signals to patient physiological outcomes. 

This portal maps deep transcriptomic signature activity scores against physiological inflammatory markers to aid in stratified, precision medicine diagnostics for complex autoimmune lineages.

🚀 **Live Application:** [View ImmunoTracks Portal](https://immunotracksautoimmune.streamlit.app/)

---

## 🛠️ Key Architectural Workspaces

### 1. ℹ️ Portal Overview & Exhibit
* **The Mission:** Introduces the translational objective of moving molecular assays from "bench to bedside."
* **Global Prevalence Trackers:** Features dynamic landscape charts visualizing the public health impacts of conditions like Rheumatoid Arthritis, Systemic Lupus (SLE), and Multiple Sclerosis (MS).

### 2. 📊 Cohort Population Analytics
* **Clinical Baselines:** Evaluates demographic variables across tracked research cohorts.
* **Dynamic Matrix Filtering:** Allows full subset scoping by age distributions, biological sex ratios, and C-Reactive Protein (CRP) scales.

### 3. 🧬 Transcriptomic Signatures
* **Pathway Alerts:** Flags molecular checkpoints (e.g., STAT1/MX1/IRF7 for Type-I Interferon pathways vs. IL6/TNF for pro-inflammatory cytokine pathways).
* **Statistical Modeling:** Renders box plots for normalized log2 expression and scatter plots displaying linear regressions against systemic inflammation indicators.
* **Co-Expression Architecture:** Features interactive Pearson correlation heatmaps to assess coordinated gene activities.

### 4. 🤖 Predictive Stratification (ML)
* **Expert Clinical Interface:** An interactive playground allowing real-time parameter tweaking.
* **Algorithmic Diagnostics:** Runs a rules-based classifier to predict disease states with dynamic biomarker rationale statements and diagnostic recommendations.

---

## 💻 Tech Stack & Engineering Core

* **Frontend Framework:** Streamlit (UI/UX deployment)
* **Data Processing:** Pandas, NumPy
* **Scientific Visualizations:** Plotly Express (Interactive charting)
* **Statistical Calculations:** Statsmodels (Ordinary Least Squares regression lines)
